# 蓝桥杯单片机 AI 辅助学习助手

## 项目简介

本项目是一个面向蓝桥杯单片机 C 组备赛学生的 AI 辅助学习项目，主要围绕 STC15F2K60S2 单片机常见模块进行代码解释、错误分析、学习笔记整理和训练题生成。

项目当前处于早期搭建阶段，主要目标是整理蓝桥杯单片机常见模块代码，并尝试通过大模型辅助理解代码逻辑、定位问题、生成注释和复盘学习过程。

## 项目背景

蓝桥杯单片机 C 组常见考点包括：

- 数码管动态扫描
- 矩阵键盘扫描
- DS18B20 温度读取
- PCF8591 / IIC 通信
- LED 状态控制
- 继电器与蜂鸣器控制
- 定时器中断
- 多界面显示与参数设置

对于初学者来说，这些模块不仅需要理解硬件连接，还需要掌握 C 语言、寄存器配置、中断逻辑和模块之间的配合。  
因此，本项目计划引入大模型能力，用于辅助学习和代码调试。

## 目标用户

本项目主要面向：

- 职业院校电子信息类学生
- 蓝桥杯单片机 C 组备赛学生
- STC15F2K60S2 初学者
- 需要理解 C51 单片机代码的学习者

## 核心功能规划

### 1. 单片机代码解释

输入一段 C51 单片机代码后，AI 辅助解释：

- 函数作用
- 变量含义
- 中断逻辑
- 外设控制流程
- 模块之间的调用关系

### 2. 错误定位与调试建议

根据代码现象或运行问题，辅助分析可能原因，例如：

- 数码管显示异常
- 按键无反应
- 温度读取错误
- IIC 通信失败
- LED 逻辑错误
- 定时器刷新异常

### 3. 学习笔记生成

将代码和模块知识整理成适合初学者理解的学习笔记，包括：

- 模块原理
- 代码结构
- 常见错误
- 调试步骤
- 备赛复盘

### 4. 训练题生成

围绕蓝桥杯常见模块生成训练任务，例如：

- 温度显示界面
- 参数设置界面
- 按键切换界面
- LED 联动逻辑
- ADC / DAC 数据处理

## 技术需求

本项目后续计划接入 Xiaomi MiMo API，用于实现以下能力：

- 长上下文代码理解
- C51 单片机代码分析
- 多文件工程代码解释
- 代码错误排查
- 学习笔记自动生成
- 蓝桥杯模块训练题生成

由于单片机项目通常包含 `main.c`、`iic.c`、`onewire.c`、`ds18b20.c` 等多个文件，因此需要模型具备较好的代码上下文理解能力。
